# Change Log

All notable changes to the "ChronoDeck" extension will be documented in this file.

## [0.0.3] - 2026-01-15

### Fixed
- **Copy Function**: Fixed clipboard copy functionality for Unix timestamps and timezone values in history cards.

## [0.0.2] - 2025-12-30

### Fixed
- **Marketplace Icon**: Fixed an issue where the extension icon was not displaying correctly in the Marketplace.

## [0.0.1] - 2025-12-30

### Released
- Initial release of ChronoDeck.
- **Features**:
    - **History Tracking**: Log current time or manual entries with ease.
    - **Timezone Support**: View timestamps in multiple timezones (e.g., UTC, JST) simultaneously.
    - **Unix Timestamp Conversion**: Automatic conversion and copy-paste support.
    - **Memo**: Add notes to each timestamp history item.
    - **Strict Input Mode**: Ensures data integrity with real-time validation and formatting.
    - **Localization**: English and Japanese support.
    - **Modern UI**: Dark-mode optimized, responsive design.
